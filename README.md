## Setup
Clone github.

Run `composer install`

Run `./vendor/bin/sail artisan migrate`

Run `./vendor/bin/sail up`


## Use

http://localhost/api/rover/init?direction=N&position_x=13&position_y=9


## Tests

Run`./vendor/bin/sail artisan test`
